### 3️⃣ 深度學習技術分析

#### 🔬 模型架構

##### LSTM + Attention 模型
```python
Input (10 features × 60 timesteps)
    ↓
Bi-LSTM (3 layers, 128 hidden units)
    ↓
Multi-head Attention (4 heads)
    ↓
Fully Connected (128 → 32 → 1)
    ↓
Price Prediction
```

##### Transformer 模型
```python
Input + Positional Encoding
    ↓
Transformer Encoder (4 layers)
    ├─ Multi-head Self-Attention (8 heads)
    ├─ Feed-forward Network
    └─ Layer Normalization
    ↓
Price Prediction
```

### 4️⃣ 量化交易策略

#### 策略 A：情感分析策略
```python
買入條件：
✓ 新聞情感 > 0.2 (正面) AND
✓ RSI < 30 (超賣) OR
✓ SMA(7) 突破 SMA(25) (黃金交叉)

賣出條件：
✓ 新聞情感 < -0.2 (負面) AND
✓ RSI > 70 (超買) OR
✓ SMA(7) 跌破 SMA(25) (死亡交叉)
```

#### 策略 B：MACD 策略
```python
買入：MACD 線上穿信號線
賣出：MACD 線下穿信號線

參數：
- 快速 EMA: 12
- 慢速 EMA: 26
- 信號線: 9
```

#### 策略 C：機器學習策略
```python
買入：預測漲幅 > 2%
賣出：預測跌幅 > 2%

使用：LSTM/Transformer 預測
```

### 5️⃣ 回測系統

#### 績效指標：

| 指標 | 說明 | 計算方式 |
|------|------|----------|
| 總回報率 | 策略累計收益 | (終值-初值)/初值 × 100% |
| 夏普比率 | 風險調整收益 | (平均收益-無風險利率)/收益標準差 |
| 最大回撤 | 最大虧損幅度 | Max((峰值-谷值)/峰值) |
| 勝率 | 盈利交易佔比 | 盈利次數/總交易次數 × 100% |
| 盈虧比 | 平均盈虧 | 平均盈利/平均虧損 |
